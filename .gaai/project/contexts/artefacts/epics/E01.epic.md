# Epic: Project Foundation & Connectivity

## Purpose

An AI assistant can launch the truenas-mcp server and establish an authenticated connection to TrueNAS SCALE. This is the foundation — nothing else works without it.

---

## Scope

- Go module initialization with fang/cobra CLI scaffolding
- `serve` command that starts an MCP server over stdio
- TrueNAS WebSocket JSON-RPC connection and API key authentication
- Configuration loading (host, API key)
- Basic MCP tool listing response

---

## Out of Scope

- Any TrueNAS domain-specific tools (handled by E02–E06)
- HTTP/SSE transport
- Multi-server configuration

---

## Stories

> Stories are tracked in the backlog. This list is descriptive only.

- E01S01: Project scaffolding with fang/cobra and serve command
- E01S02: TrueNAS WebSocket connection and authentication
- E01S03: MCP server initialization with stdio transport

---

## Success Metrics

- `truenas-mcp serve` starts an MCP server that responds to `tools/list`
- Server connects and authenticates to TrueNAS via WebSocket
