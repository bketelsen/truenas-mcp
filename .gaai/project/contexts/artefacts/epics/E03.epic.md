# Epic: Storage Management

## Purpose

An AI assistant can manage storage resources — creating and deleting datasets and snapshots — enabling automated storage maintenance workflows.

---

## Scope

- Create and delete datasets
- Create and delete snapshots
- Update dataset properties

---

## Out of Scope

- Pool creation/destruction (dangerous, low-frequency, better done in UI)
- Replication configuration

---

## Stories

> Stories are tracked in the backlog. This list is descriptive only.

- E03S01: Create and delete datasets
- E03S02: Create and delete snapshots

---

## Success Metrics

- AI assistant can create a dataset, snapshot it, and clean up old snapshots through MCP tools

---

## Dependencies

E01 (foundation), E02 (visibility tools for verification)
