# Epic: File Sharing Access

## Purpose

An AI assistant can view and manage SMB and NFS shares, enabling network share administration without accessing the TrueNAS UI.

---

## Scope

- List SMB and NFS shares
- Create, update, and delete SMB shares
- Create, update, and delete NFS shares

---

## Out of Scope

- iSCSI configuration
- Share permission/ACL management beyond basic share-level settings

---

## Stories

> Stories are tracked in the backlog. This list is descriptive only.

- E04S01: List and inspect SMB and NFS shares
- E04S02: Create, update, and delete shares

---

## Success Metrics

- AI assistant can list all shares and create a new SMB share through MCP tools
