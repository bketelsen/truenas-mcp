# Epic: System Awareness

## Purpose

An AI assistant can monitor TrueNAS system health, view alerts, and check disk status — enabling proactive NAS monitoring and troubleshooting.

---

## Scope

- System information (version, uptime, hostname)
- Disk health and status
- Alert listing and dismissal
- Basic network interface info

---

## Out of Scope

- System configuration changes (reboot, shutdown, network reconfiguration)
- Update management

---

## Stories

> Stories are tracked in the backlog. This list is descriptive only.

- E05S01: System information and disk health
- E05S02: Alert listing and management

---

## Success Metrics

- AI assistant can report system health status and surface active alerts through MCP tools
