# Epic: App Management

## Purpose

An AI assistant can view and control installed apps (Docker containers) on TrueNAS, enabling application lifecycle management.

---

## Scope

- List installed apps with status
- Query app details and resource usage
- Start, stop, and restart apps

---

## Out of Scope

- App installation and configuration (complex, better done in UI)
- Custom Docker compose deployment

---

## Stories

> Stories are tracked in the backlog. This list is descriptive only.

- E06S01: List and inspect apps
- E06S02: App lifecycle management (start/stop/restart)

---

## Success Metrics

- AI assistant can list running apps and restart a specific app through MCP tools
