# Story: List and inspect pools

## User Story

**As an** AI assistant user,
**I want** to ask the AI to show me my ZFS pools and their status,
**so that** I can understand my storage topology at a glance.

---

## Context

Exposes `pool.query` via MCP tools. Pools are the top-level storage construct in TrueNAS.

---

## Acceptance Criteria

- [ ] AC1: A `truenas_pool_list` MCP tool exists that returns all pools with name, status, size, and health
- [ ] AC2: A `truenas_pool_get` MCP tool exists that returns detailed info for a specific pool by name or ID
- [ ] AC3: Results include topology information (vdevs, disks)

---

## Out of Scope

- Pool creation or modification
- Scrub scheduling

---

## Dependencies

- E01S03 (MCP server with TrueNAS connection)

---

## Notes

Use `pool.query` with appropriate filters and select fields to avoid returning excessive data.
