# Story: App lifecycle management (start/stop/restart)

## User Story

**As an** AI assistant user,
**I want** to ask the AI to start, stop, or restart apps on my NAS,
**so that** I can manage services without opening the TrueNAS UI.

---

## Context

Exposes app lifecycle methods via MCP tools. These are common maintenance operations.

---

## Acceptance Criteria

- [ ] AC1: A `truenas_app_start` tool starts a stopped app by name
- [ ] AC2: A `truenas_app_stop` tool stops a running app by name
- [ ] AC3: A `truenas_app_restart` tool restarts an app by name
- [ ] AC4: All tools return the resulting app status after the operation

---

## Out of Scope

- App installation, removal, or configuration
- Bulk operations across multiple apps

---

## Dependencies

- E06S01 (app visibility for verification)

---

## Notes

Lifecycle operations may take time. Tools should wait for completion or report the job status.
