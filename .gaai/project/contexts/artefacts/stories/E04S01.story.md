# Story: List and inspect SMB and NFS shares

## User Story

**As an** AI assistant user,
**I want** to ask the AI to show me all configured network shares,
**so that** I can understand what data is accessible on the network.

---

## Context

Exposes `sharing.smb.query` and `sharing.nfs.query` via MCP tools.

---

## Acceptance Criteria

- [ ] AC1: A `truenas_smb_list` MCP tool exists that returns all SMB shares with name, path, and enabled status
- [ ] AC2: A `truenas_nfs_list` MCP tool exists that returns all NFS exports with path, networks, and enabled status
- [ ] AC3: Both tools return share-specific settings (e.g., guest access for SMB, maproot for NFS)

---

## Out of Scope

- Share creation/modification (E04S02)
- ACL management

---

## Dependencies

- E01S03 (MCP server with TrueNAS connection)
