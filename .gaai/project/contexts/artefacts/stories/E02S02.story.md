# Story: List and inspect datasets

## User Story

**As an** AI assistant user,
**I want** to ask the AI to list my datasets and their properties,
**so that** I can understand storage usage and configuration.

---

## Context

Exposes `pool.dataset.query` via MCP tools. Datasets are the primary organizational unit for data in ZFS.

---

## Acceptance Criteria

- [ ] AC1: A `truenas_dataset_list` MCP tool exists that returns datasets with name, used space, available space, mountpoint, and compression
- [ ] AC2: Tool supports an optional `pool` parameter to filter datasets by pool
- [ ] AC3: A `truenas_dataset_get` MCP tool exists that returns full properties for a specific dataset by path

---

## Out of Scope

- Dataset creation or deletion (E03S01)
- Quota management

---

## Dependencies

- E01S03 (MCP server with TrueNAS connection)

---

## Notes

Dataset queries can return large result sets on busy NAS servers. Consider pagination or filtering by pool.
