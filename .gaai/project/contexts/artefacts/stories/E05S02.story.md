# Story: Alert listing and management

## User Story

**As an** AI assistant user,
**I want** to ask the AI about active alerts on my NAS,
**so that** I can quickly identify and address issues.

---

## Context

Exposes `alert.list` and `alert.dismiss` via MCP tools. Alerts are TrueNAS's built-in monitoring system for hardware, software, and configuration issues.

---

## Acceptance Criteria

- [ ] AC1: A `truenas_alert_list` tool returns all active alerts with level (info/warning/critical), message, datetime, and dismissed status
- [ ] AC2: A `truenas_alert_dismiss` tool dismisses an alert by ID
- [ ] AC3: Alert list tool supports filtering by level (e.g., only critical alerts)

---

## Out of Scope

- Alert configuration (notification settings, thresholds)
- Alert creation

---

## Dependencies

- E01S03 (MCP server with TrueNAS connection)

---

## Notes

Alerts are read-heavy. Dismiss is the only mutating operation.
