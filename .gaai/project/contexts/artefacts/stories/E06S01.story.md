# Story: List and inspect apps

## User Story

**As an** AI assistant user,
**I want** to ask the AI what apps are running on my NAS,
**so that** I can monitor my self-hosted services.

---

## Context

TrueNAS SCALE Goldeye uses Docker for app management. Exposes app query methods via MCP tools.

---

## Acceptance Criteria

- [ ] AC1: A `truenas_app_list` tool returns all installed apps with name, version, status (running/stopped), and update availability
- [ ] AC2: A `truenas_app_get` tool returns detailed info for a specific app by name

---

## Out of Scope

- App installation or removal
- App configuration changes
- Docker compose management

---

## Dependencies

- E01S03 (MCP server with TrueNAS connection)

---

## Notes

App API surface may vary between TrueNAS versions. Goldeye uses Docker-based apps (migrated from Kubernetes in earlier releases).
